## 四、结果分析

### 4.1 异常值检测

**实验9出现严重异常**，误差是其他正常实验的3-7倍：

| 指标 | 实验9 | 正常实验平均 | 倍数 |
|------|-------|-------------|------|
| 测试PEHE | 10.377 | ~0.86 | 12x |
| 训练PEHE | 14.267 | ~0.95 | 15x |
| 测试ITE | 4.538 | ~1.09 | 4x |

**可能原因**：
- 随机初始化落入局部最优解
- 梯度爆炸/消失问题
- 数值不稳定

## 六、结论与建议

### 6.1 成功复现
- ✅ 代码成功运行并完成10次重复实验
- ✅ 9/10的实验表现良好
- ✅ 模型架构和训练流程正确
- ✅ 结果与原论文基本一致

### 6.2 存在问题
1. **训练不稳定**：约10%的实验（实验9）出现严重偏差
2. **标准差较大**：说明模型对初始化敏感

### 6.3 改进建议

| 建议 | 优先级 | 说明 |
|------|--------|------|
| 增加实验次数 | 高 | 建议50-100次以获得更稳定的统计结果 |
| 梯度裁剪 | 高 | 防止梯度爆炸 |
| 学习率衰减 | 中 | 可能提高收敛稳定性 |
| 多随机种子 | 中 | 使用多个随机种子验证结果 |
| 超参数调优 | 低 | 当前超参数可能不是最优 |

### 6.4 运行命令

```bash
# 激活环境
conda activate cevae

# 运行实验（默认10次重复）
python cevae_ihdp.py

# 自定义重复次数
python cevae_ihdp.py -reps 50

# 调整学习率
python cevae_ihdp.py -lr 0.0005

# 调整训练轮数
python cevae_ihdp.py -epochs 150
```
